Empty Element

Empty Elements are elements that do not have an element content and an end tag.
The < br / > tag defines a line break, and is an empty element without a Closing tag:

Output:

This is a
Paragraph with line Break

A list of Commonly used Empty Elements:

  1. <meta />
  2. <link />
  3. <img />
  4. <hr />
  5. <input />

The best practice in HTML Empty HTML Empty Elements is to always put a forward slash / sign before the greater than > sign

in this way they are closed at their start tags. These tags are also called self-closing tags.

HTML is NOT Case Sensitive

HTML tags is not case sensitive:<P>means the same as <p>.<H1>
means the same as <h1>.

The HTML standars does not require lowercae tags,but W3C recommends
lowercase in HTML,and demands lowercase for stricter document types like
XHTML.

We will always use lowercase tag names.
