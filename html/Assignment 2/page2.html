Attributes

The HTML attributes are used to provide more information to an HTML element.

  • HTML attributes only appear at start tags. it will never be on end tags.
  • All HTML elements can have multiple attirbutes.
  • HTML attributes usually come in name/value pairs like:name = "value"
    • an Attributes name
    • an equal = sign
    • A value surrounded by Quotations marks "value"

you can also use single quotation marks depending on the situation esp.
when the value contains double quotes.

For you to better understand it look at the example code below.

<p tital="I serve as a tooltip">This is a paragraph</p>
<p tital="I am a tooltip">This is a paragraph</p>
<p tital='It's going to break'>This is a paragraph</p>

<a href="login.html">Login</a>
<a href="https://umairahmad.net" target-"_blank">Muhammad Umair Ahmad</a>

There are two ways to specify the URL in the href attribute

  1. Absolute URL:Links to an External website that is hosted on another
    server.
  2. Relative URL:Links to an html page that is hosted within the website.
    Here, the URL does not include the domain name.If the URL begins
    without a slash, it will be relative to the current. Example:
    href="login.html".
